# ARIA_Hackathon_2026
Projeto em que desenvolvemos é uma plataforma de Inteligência Regulatória baseada em IA Generativa que automatiza a ingestão, interpretação e o monitoramento contínuo de documentos científicos e normas técnicas. A solução transforma bases de dados complexas e estáticas em um "cérebro corporativo" vivo e consultável em linguagem natural.
